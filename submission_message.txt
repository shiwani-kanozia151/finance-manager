## Setup Instructions:
1. Clone the repository
2. Run `npm install` to install dependencies
3. Copy `.env.example` to `.env` and configure environment variables
4. Run `npx prisma migrate dev` to set up the database
5. Run `npx prisma generate` to generate Prisma client
6. Start the server with `npm start`

The API provides endpoints for user registration/login, financial record management, and dashboard analytics, all protected with JWT authentication.
